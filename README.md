# GoalCraft

GoalCraft is an agent-assisted goal planning system that helps users turn high-level goals into actionable tasks and adapt plans based on progress.

## Problem Statement
People often set ambitious goals but struggle to break them into clear steps and stay consistent over time.

## Solution
GoalCraft introduces a reasoning layer that dynamically plans, tracks, and adjusts tasks instead of relying on static to-do lists.

## Core Features (Planned)
- Create and manage long-term goals
- Automatic task breakdown
- Progress tracking
- Adaptive task re-planning

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Agent Logic: Rule-based (extendable to AI)
